# 0x02-shell_redirections: I/O Redirection and Filters This directory
contains solutions for the tasks related to Standard I/O streams and
filtering data in the Linux shell. ## Key Concepts Covered: * Standard
Input (0), Standard Output (1), Standard Error (2) * Redirection Operators
(`>`, `>>`, `<`) * Piping data between commands (`|`) * Using Filters
(`cat`, `head`, `tail`, `sort`, `uniq`)
